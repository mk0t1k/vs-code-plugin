# Документация проекта TEXT_CONVERTER

## Общее описание решения

Данный проект предоставляет расширение для Visual Studio Code, позволяющее быстро преобразовывать выделенный текст между различными стилями написания. Расширение реализовано на TypeScript и состоит из основного модуля, который обрабатывает команды и применяет соответствующие функции преобразования.

### Структура проекта

- `src/extension.ts` - основной модуль с функциями расширения
- `package.json` - манифест расширения VS Code
- `tsconfig.json` - конфигурация TypeScript
- `docs/` - директория с документацией

## Описание функций

### Модуль extension.ts

#### `activate(context)`
Активирует расширение при его загрузке в VS Code.

**Параметры:**
- `context` (`vscode.ExtensionContext`): контекст расширения VS Code

**Возвращаемое значение:**
- `void`

#### `deactivate():`
Деактивирует расширение при его выгрузке. В текущей реализации дополнительных действий не выполняет.
**Параметры:**
- нет

**Возвращаемое значение:**
- `void`
